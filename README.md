## 介绍
采用AMD模块化，用requirejs加载js，gulp构建项目，bootstrap提供ui支持
gulp插件：
	gulp-sass
	gulp-imagemin
	gulp-uglify
	gulp-clean
	gulp-jshint
	browser-sync
src: 源文件
## 开始
1.安装
```bash
npm install
bower install
```
2.开始任务
```bash
gulp
```
